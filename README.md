mikutter_oshaburi
=====================

なにこれ
---

ローカルファイルにツイートを書き出すプラグインです。わりとコンセプトモデルに近いので、だいぶ荒削りです。2014年の新春フライト中に書きました（どうでもいい）。

+ インターネットが繋がらない環境下でもmikutterでひたすらツイートっぽいことができる
+ とっさのメモ書きなどに便利かも


なんでおしゃぶりなの
---

四六時中ツイートしてないと死ぬ奴にとってこいつぁおしゃぶりみたいなものなんだぜ？


つかいかた
---

1. プラグインをインストロールします
2. 設定画面の「ローカルツイート」で、保存先ディレクトリを指定します
3. 適宜ショートカットキーを割り当てます
4. 保存先ディレクトリにローカルツイートされれば成功です
